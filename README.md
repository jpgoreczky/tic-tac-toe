# Tic Tac Toe Game

## Description
This is a simple Tic Tac Toe game implemented using HTML, CSS, and JavaScript. It allows two players to play against each other on a grid of 3x3 squares. The players take turns marking the spaces with "X" or "O" until one player wins by having three of their marks in a horizontal, vertical, or diagonal row, or until the board is full and the game ends in a draw.

## Features
- Interactive gameplay that allows two players to take turns making moves.
- Visual indication of whose turn it is and which squares have been marked by each player.
- Win detection algorithm to determine the winner or if the game ends in a draw.
- Reset button to start a new game after the current game is finished.

## Technologies Used
- HTML: Provides the structure and layout of the game board.
- CSS: Styles the game board, squares, and other elements for a visually appealing interface.
- JavaScript: Implements the game logic, including player turns, win detection, and reset functionality.

## Usage
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. Start playing by clicking on an empty square to place your mark ("X" or "O").
4. Take turns with your opponent until one player wins or the game ends in a draw.
5. Click the "Reset" button to start a new game after the current game is finished.
